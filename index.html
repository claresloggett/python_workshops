Introductory Python: Python for Bioinformaticians

These workshops are designed to help people involved in bioinformatics to learn programming, using Python. They are based around implementing some simple bioinformatics algorithms.

These workshop borrow heavily from Software Carpentry. You can find a the comprehensive set of Software Carpentry lessons that we have adapted some content from here.

Contents

  1. Setup
  2. Introductory Python - lists, loops and strings: Hamming Distance
  3. More Python - dictionaries, sets, and slicing: K-mer Counting
  4. Extra stuff: Libraries

Detailed setup instructions

If you'd like to install a full Python distribution on your own computer, you can find good instructions in the Software Carpentry setup guide.
